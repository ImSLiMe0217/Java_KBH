### 데이터 타입

기본형(Primitive Type)

1\) 정수형

\* : 자바 기본타입

        byte        : 1bite (8 bits), -128 ~ 127 범위 (총 256개 (2^8))
        short       : 2bite (16 bits), -32768 ~ 32767 범위 
        * int       : 4bite (32 bits), 약 -21억 ~ 21억 
        long        : 8bite (64 bits), 약 -9.22E18 ~ 9.22E18, 선언 시 숫자 끝에 l을 붙여서 선언

2\) 실수형

\* : 자바 기본타입


        flaot       : 4byte (32 bits), 유효 숫자 7자리, 선언시 숫자 끝에 f를 붙여서 선언
        * double    : 8byte (64 bits), 유효 숫자 15자리

3\) 논리형과 문자형

        boolean     : 1byte (8 bits), true / false 값만 가짐
        char        : 2byte (16 bits), 문자 1개만 저장, 작은따옴표(')로 감싸서 선언

### 변수와 상수

변수(Variable)
* 데이터를 담는 그릇
* 변수에 값을 저장, 그 값이 빌요할 때 변수를 호출해서 사용
* 값은 언제든지 수정가능
* 다른 변수와 식별하기 위한 고유의 이름(식별자)을 가짐

        변수 선언

            (데이터형) (변수명) = (값)
            ex) int age = 26;
