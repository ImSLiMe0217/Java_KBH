### Package와 Import

- Package: 관련된 클래스들을 그룹화하는 폴더 구조
- Import: 다른 패키지에 있는 클래스를 사용할 떄 명시(java.lang 패키지는 예외)

```java
// 패키저 선언 (클래스의 실제 디렉토리 경로와 매핑) 
package example;

// 외부 패키지에 포함된 java.util.Scanner 클래스 임포트

import java.util.Arrays;

public class example1 {
    static void main() {
        int arr = {1, 2, 3, 4};

        System.out.println(Arrays.toString(arr)); // [1, 2, 3 ,4]
    }
}
```

### 캡슐화와 접근 제어자

- 캡슐화: 중요한 데이터와 이를 처리하는 메서드를 하나로 묶고, 외부에서 임의로 변경하지 못하도록 데이터를 은닉
- 접근제어자: private < default < protected < public 순으로 접근 권한을 제한
- 필드를 private으로 선언하고 Getter/Setter 및 유효성 검증 메서드를 통해서만 접근하도록 제한

#### 접근 제어자별 세부 접근 범위

| 접근 제어자 클래스  | 동일 클래스 | 동일 패키지 | 동일 패키지 및 상속관계 | 어디서나   |
|-------------|--------|--------|---------------|--------|
| `public`    | 허용 (O) | 허용 (O) | 허용 (O)        | 허용 (O) |
| `protected` | 허용 (O) | 허용 (O) | 허용 (O)        | 차단 (X) |
| `default`   | 허용 (O) | 허용 (O) | 차단 (X)        | 차단 (X) |
| `private`   | 허용 (O) | 차단 (X) | 차단 (X)        | 차단 (X) |

#### Getter / Setter

- 클래스 외부에서 private 필드에 안전하게 접근하고 값을 변경하기 위해 사용
- 작성규칙
    - Getter: get + 필드명 (첫 글자 대문자) (ex: String name -> getName)
    - Setter: set + 필드명 (첫 글자 대문자) (ex: setName(String name))
- 논리형 필드 예외규칙
    - Getter: get이 아닌 is로 시작
    - Setter: 일반적인 작성규칙과 동일

### 정적(static) 멤버

- static 키워드가 붙은 필드나 메서드는 인스턴스 생성을 하지 않아도 클래스명으로 바로 접근 가능
- 모든 인스턴스가 공유하는 메모리(메서드 영역)에 저장됨
- static 메서드의 제한 사항:
    - static 메서드 내부에서는 static이 아닌 필드나 메서드를 직접 불러올 수 없음
    - static 메서드 내부에서는 오직 static 필드나 static 메서드만 직접 호출 가능
    - this 키워드는 static 메서드 내부에서는 사용할 수 없음

```java
public class MathUtils {
    public static final double PI = 3.141592;

    public static int add(int a, int b) {
        return a + b;
    }

    public int sum(int[] arr) {
        int sum = 0;
        for (int elem : arr) {sum += elem;}
        return sum;
    }
}

public class Test {
    static void main() {
        int result1 = MathUtils.add(3, 4);
        System.out.println(result1); //int result2 =  7

        int[] arr = {1, 2, 3, 4, 5};
        int result2 = MathUtils.sum(arr);   // 컴파일에러 
        // -> static 메서드인 main메서드에서 static이 아닌 sum메서드는 호출할 수 없음

        MathUtils mu = new MathUtils();
        int result3 = mu.sum(arr);  // 이런 식으로 호출해서 사용해야함
    }
}
```
