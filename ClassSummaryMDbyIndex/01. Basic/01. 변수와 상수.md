# Java 기초
___
## 1. 변수와 상수
___
## 변수 (Variable)
- 값을 저장하는 그릇
- 8개의 기본형과 참조형으로 구분됨

### 기본형
___
- 가장 기본이 되는 변수타입
- 정수형 4개, 실수형 2개, 문자형, 논리형 총 8개가 존재
  - 정수형: byte, short, int, long
  - 실수형: float, double
  - 문자형: char
  - 논리형: boolean
- stack 메모리 영역에 저장됨
- 변수의 타입에 따라 메모리의 크기가 달라짐

#### 정수형
    - byte
        1byte(8bits)의 크기를 가짐

    - short
        2byte(16bits)의 크기를 가짐

    - int
        4byte(32bits)의 크기를 가짐, 
        정수 선언 시 기본적으로 사용되는 자료형

    - long
        8byte(64bits)의 크기를 가짐, 
        선언시 숫자 옆에 'L' 또는 'l'을 붙여 선언

#### 실수형
    - float
        4byte(32bits)의 크기를 가짐
        유효숫자 7자리(정수부 포함)
        선언 시 숫자 뒤에 'F'나 'f'를 붙여 선언

    - double
        8byte(64bits)의 크기를 가짐
        유효숫자 15자리(정수부 포함)
        실수 선언 시 기본적으로 사용하는 자료형

#### 문자형
    - char
        2byte(16bits)의 크기를 가짐
        1개의 문자를 저장할 때 사용하는 자료형
        ASCII코드나 UNI코드도 사용 가능

#### 논리형
    - boolean
        1byte(8bits)의 크기를 가짐
        true 혹은 false 두 가지의 값 중 하나를 저장
        
```java
void main() {
    // 정수형
    byte b = 2^8 - 1;
    short s = 2^16 - 1;
    int i = 2^32 - 1;
    long l = 2^64 - 1 ;
    
    // 실수형
    float f = 1.234567f;
    double d = 1.23456789012345;
    
    // 문자형
    char c = 'A';
    
    // 논리형
    boolean bool = true; // 또는 false
}
```

### 참조형
___
- 기본형 8개를 제외한 모든 변수타입을 참조형이라함
- 예시: String, 배열, 클래스 등
- 값은 Heap메모리 영역에 저장하고 stack에는 값이 저장된 Heap메모리의 주소값을 저장
- 참조형 변수의 크기는 내부 값에 상관 없이 항상 일정한 크기를 가짐

## 상수
- 고정된 값
- 선언 이후 값 재할당 불가능
- 변수타입 앞에 `final`을 붙여서 선언
- 예시:
```java
final double PI = 3.14;
```

## 식별자
- 변수 또는 상수의 이름
- 식별자 생성 규칙을 따름
- 식별자 생성 규칙:
      
      1. 알파벳 대소문자와 숫자, 밑줄(_)과 달러기호($)만 사용 가능
      2. 숫자로 시작할 수 없음
      3. 예약어(int, if, while, return 등)는 사용할 수 없음
      4. 변수나 메소드의 식별자는 영어 소문자로 시작해서 단어가 바뀔 때 첫글자를 대문자로 사용
      5. 상수의 식별자는 모두 대문자로 사용 단어가 바뀔 때는 밑줄(_)을 사용하여 구분
      6. 클래스의 식별자는 대문자로 시작하고 단어가 바뀔 때에도 첫글자를 대문자로 시작

- 예시:
```java
public class Test { // Test 클래스
    final double PI = 3.14;     // 상수
    final int INIT_SCORE = 0;   // 상수 (2단어)
    void main() {   // main 메소드
        int score = INIT_SCORE;     // 변수 (1단어)
        int totalScore = 100;       // 변수 (2단어)
    }
}
```
