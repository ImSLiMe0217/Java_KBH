### 2. 식별자와 예약어

#### 2.1 식별자
- 변수, 상수, 메서드, 클래스 등에 붙여주는 '이름'
- 개발자가 임의로 지정할 수 있지만 지켜야 할 문법적 규칙과 관례가 존재
  
      식별자 명명 규칙
      - 영문 대소문자, 숫자, 밑줄(_), 달러 기호($)만 사용가능
      - 숫자로 시작할 수 없음
      - 예약어는 사용할 수 없음


      식별자 명명 관례
      - 클래스 이름: 대문자로 시작
      - 변수와 메소드: 소문자로 시작, 단어가 바뀌면 대문자 사용
      - 상수 이름: 모두 대문자로 작성, 단어 사이는 밑줄(_) 사용해서 구분

#### 2.2 예약어
- 자바에서 미리 점유해서 사용하고 있는 단어
- 식별자로 사용할 수 없음
- 예: `int`, `if`, `for`, `true`, `false`

### 3. 형변환
서로 다른 타입끼리 연산하거나 대입할 떄 크기를 맞춰주는 작업

#### 3.1 자동형변환 (묵시적 형변환)
- 작은 크기의 타입을 큰 크기의 타입으로 변환(데이터 손실 없음)
- `byte` -> `short` -> `int` -> `long` -> `float` -> `double`

#### 3.2 강제형변환 (명시적 형변환)
- 큰 크기의 타입을 작은 크기의 타입으로 강제로 변환 (데이터 손실 위험)
- 식별자 앞에 (변수타입)을 명시해 줘야함
- 예시
```java
void main () {
  int i = 10;
  double pi = 3.14;
  int diff = i - (int)pi; // = 7
}
  ```

### 4. 연산자

#### 4.1 산술연산자
- `+`, `-`, `*`, `/`, `%`(나머지 연산자)
- 결합연산자(+): 문자열과 함께 사용할 경우 문자열 결합을 의미
- 예시
```java
void main () {
  int age = 30;
  String str = "나이는" + age  + " 입니다.";
  System.out.println("str");    // "나이는 30 입니다."
}
```

#### 4.2 대입연산자
- `=`, `+=`, `-=`, `*=`, `/=`, `%=`
- 우측의 값을 계산한 후 왼쪽에 대입
- 예시
```java
void main () {
  int a = 10 + 5;   // a = 15
  int sum = 20;
  sum += a;         // sum = sum + a; 하고 동일한 결과
}
```
  
#### 4.3 증감연산자
- `++`, `--`
- 값을 1 증가 또는 감소
- 변수 앞에 붙으년 연산 전 증감(전위연산), 뒤에 붙으면 연산 후 증감(후위연산)
```java
void main () {
  int a = 10;
  int b = ++a;    // 전위 연산: a값 증가시킨 후 b에 대입
  System.out.println("a: " + a + ", b: " + b);  // a = 11, b: 11
  
  int x = 10;
  int y = x++;    // 후위 연산: x의 현재 값을 먼저 y에 대입 후 x 값을 증가
  System.out.println("x: " + x +", y: " + y);  // x = 10, y: 11   
}
```

#### 4.4 비교연산자
- `==`, `!=`, `>`, `<`, `>=`, `<=`
- 두 항을 비교하여 참이면 true, 거짓이면 false를 반환

#### 4.5 논리연산자
- `&&`(and), `||`(or), `!`(not)
- `&&`: 두 항이 모두 참이면 true, 그 외는 false를 반환
- `||`: 두 항이 모두 거짓으면 false 그 외는 true를 반환
- `!`: 단항연산자, 우측 항이 참이면 false, 거짓이면 true를 반환

#### 4.6 삼항연산자
- `조건식 ? 참일 때 값 : 거짓일 때 값`
- 예시
```java
void main () {
  int score = 85;
  String result = (score >= 80) ? "합격" : "불합격";
  System.out.println("result"); // 합격 
}
```
  
#### 4.7 연산자 우선순위
0. 괄호 `(), []`
1. 산술연산자 
2. 비교연산자 
3. 논리연산자 
4. 대입연산자
