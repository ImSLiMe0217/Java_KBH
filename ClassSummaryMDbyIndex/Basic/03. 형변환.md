# 자바 기초

## 3. 형변환
___
- 연산 과정에서 사용되는 모든 변수들은 같은 타입을 가지고 있어야 함
- 서로 다른 타입의 변수들로 연산을 수행할 때는 `형변환`을해주어야 함 

### 묵시적형변환 (자동형변환)
- 크기가 작은 자료형은 크기가 큰 자료형으로 자동형변환이 가능
- `byte` -> `short` -> `int` -> `long` -> `float` -> `double` -> `String`(문자열)
- 예시:
```java
void example1() {
    int score1 = 100, score2 = 90, score3 = 75;
    int sum = score1 + score2 + score3;
    
    // int와 double간의 연산
    double avg = sum / 3.0; // sum이 double로 자동형변환
}
```

### 암시적형변환 (강제형변환)
- 크기가 큰 자료형을 강제적으로 크기가 작은 자료형으로 형변환
- 데이터 손실 가능성 있음
- 식별자 앞에 (변환할 타입)형태를 붙여서 사용
- 예시:
```java
void example2() {
    final double PI = 3.14;
    
    // double -> int로 형변환
    int pi = (int) PI;
    System.out.println(pi); // 3 -> 소수점 자리 데이터 손실
}
```
