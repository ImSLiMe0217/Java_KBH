### switch 문
- 각 case에 맞는 상황일 때만 실행하게 만드는 코드
- 조건문이 참이 되는 범위가 좁거나 1개 씩 띄워진 상황에서 주로 사용

### 기본형태
```markdown
switch (매개변수) {
    case (매개변수가 가질 수 있는 값): 
        실행할 구문
        break;
    default:
        일치하는 경우가 없을 경우 실행할 구문
} 형태 
```
    
```java
void example() {
    int floor = 2;
    switch (month) {
        case 1:
            System.out.println("1층");
            break;
        case 2:
            System.out.println("2층");
            break;
        case 3:
            System.out.println("3층");
            break;
        default:
            System.out.println("맞는 경우 없음");
    }
}
```
