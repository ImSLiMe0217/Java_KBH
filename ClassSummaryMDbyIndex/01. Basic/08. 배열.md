## 08. 배열

- 같은 타입의 데이터를 묶어서 효율적으로 관리하는 데이터 구조
- 데이터 타입 뒤에 대괄호(`[]`)를 붙여서 배열임을 명시
- 배열 내의 값을 `원소`라고 함

___

### 기본형태

```java
void format() {
    int[] 변수명 = new int[] {1, 2, 3, 4}; // 배열 선언 및 생성, 초기화
    int[] 변수명 = new int[4];             // 배열 선언 및 생성

    int[] 변수명;                          // 배열 선언
    변수명 = new int[] {1, 2, 3, 4};       // 배열 생성 및 초기화
    변수명[0] = 100;                       // 배열 내부 값 접근 및 수정
}
```

- 선언과 동시에 생성과 초기화를 수행할 경우 new 연산자를 사용하지 않아도 문제 없음
- 예시

```java
void example() {
    int[] 변수명 = {1, 2, 3, 4};
}
```

### 얕은 복사와 깊은 복사

- 얕은 복사
    - 기본형 변수처럼 식별자 만을 이용해서 배열을 복사하는 방식
    - Heap영역에서 새로운 공간을 할당해서 복사하지 않음
    - Stack영역에서만 배열의 Heap영역 주소를 복사해서 저장
    - 한 쪽에서 값을 변경하면 다른 쪽에서도 반영됨

```java
void example2() {
    int[] arr1 = {1, 2, 3, 4};
    int[] shallow = arr1;

    shallow[0] == 100;

    System.out.println(arr[0], shallow[0]); // 100, 100
}
```

- 깊은 복사
    - 기존 배열과 똑같이 생긴 새로운 배열을 생성
    - 한 쪽에서 값을 변경해도 다른 한 쪽에는 반영되지 않음
    - 다양한 방식이 존재

```java
void example3() {
    int[] arr1 = {1, 2, 3, 4};

    int[] deep1 = arr1.clone(); // 배열 객체 내장 메서드
    int[] deep2 = Arrays.copyOf(arr1, 4); // Arrays 클래스 내장 메서드

    // 단순 for문 이용
    int[] deep3 = new int[arr1.length];
    for (int i = 0; i < arr1.length; i++) {deep3[i] = arr1[i];}

    arr1[1] = 20;
    System.out.println(arr1[1], deep1[1], deep2[1], deep3[1]); // 20, 2, 2 ,2
}
```

### Arrays 클래스

- 배열 객체를 다루는 다양한 메서드를 모아놓은 클래스
- 자주 사용하는 메서드
    - Arrays.sort(대상배열): 배열 내 원소를 오름차순으로 정렬
    - Arrays.toString(대상배열): 배열의 형태와 내부 원소들을 문자열 형태로 출력
    - Arrays.copyOf(대상배열, 복사할 길이): 배열의 깊은 복사를 할 때 사용
