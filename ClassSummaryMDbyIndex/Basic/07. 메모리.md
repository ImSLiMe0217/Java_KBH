## 07. 메모리

- 자바의 각 데이터는 특정한 메모리공간에 보관됨
- 크게 Stack, Heap, Method 영역으로 구분됨
- Method 영역은 이후 Method 단원에서 다시 다룸

___

### Stack 영역과 Heap 영역

- Stack: 기본 타입 데이터의 `값`과 참조 타입 데이터의 `주소값`을 저장하는 영역
- Heap: 참조 타입 데이터의 `값`을 저장하는 영역

```java
void example() {
    int a = 10; // stack영역에 저장
    String b = new String("Hello"); // "Hello"는 Heap영역에 저장, stack에는 "Hello"의 주소값을 저장

}
```
