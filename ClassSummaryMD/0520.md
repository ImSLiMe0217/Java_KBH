## 복습

### 1. 형변환
- 자바에서의 모든 연산은 서로 같은 자료형(변수타입)끼리 이루어진다.
- 자동형변환: 용량이 작은 자료형에서 큰 자료형으로 형변환
- 강제형변환: 데이터 손실을 감수하고 큰 자료형을 작은 자료형으로 강제로 형변환

### 2. 연산자
- 종류: 산술연산자, 대입연산자, 증감연산자, 비교연산자, 논리연산자
- 연산에는 우선 순위가 있음
  - 괄호 `(), []`
  - 산술연산자
  - 비교연산자
  - 논리연산자
  - 대입연산자

## 0520 진도

### 1. 조건문

#### 1.1 if문
- 조건식이 참(true)일 때만 블록을 실행, 거짓(false)이면 실행하지 않고 건너뜀
- 형식:
```java
void main () {
  if (조건식) {
      // 조건식이 참일 때 실행할 구문
          }   
}
```

#### 1.2 if-else문
- 조건식이 참(true)일 경우 해당 구문을 수행, 거짓일 경우 else의 구문을 수행
- 형식:
```java
void main () {
  if (조건식) {
      // 조건식이 참일 때 실행할 구문
          } else {
      // 조건식이 거짓일 때 실행할 구문
          }   
}
```

#### 1.3 if-else if 문
- 조건식이 참(true)일 경우 해당 구문을 수행, 거짓일 경우 else if문의 조건식을 순차적으로 비교하여 참에 해당하는 구문을 수행
- 형식:
```java
void main () {
  if (조건식1) {
      // 조건식1이 참일 때 실행할 구문
          } else if (조건식2) {
      // 조건식2가 참일 때 실행할 구문
          } else if (조건식3) {
      // 조건식3이 참일 때 실행할 구문
          } else {
      // 모든 조건식이 거짓일 때 실행할 구문
          }   
}
```

#### 1.4 {}생략
- 조건문 뒤에 실행해야할 구문이 단 1줄이라면 중괄호({})를 생략할 수 있음
- 가능하면 그냥 생략하지 말고 사용할 것

### 2. switch문

#### 2.1 기본 형태
- 여러 줄의 else if 대신 사용가능
- 형식: 
```java
void main () {
  switch (매개변수) {
      case 변수 값의 경우의 수:
          // 해당 케이스에서 실행할 구문
      default:
          // 지정하지 않은 케이스에서 실행되는 구문
          }
}

```
- default는 없어도 됨

#### 2.2 추가된 문법
- 다중 조건 결합: 쉼표(,)를 사용하여 여러 case조건을 한 줄에 묶어 선언
- 예시: 
```java
void main () {
  int month = 5
  switch (month) {
      case 1, 3, 5, 7, 8, 10, 12:
          System.out.println("마지막 날은 31일 입니다.");
      case 4, 6, 9, 11:
          System.out.println("마지막 날은 30일 입니다.");
      case 2: 
          System.out.println("마지막 날은 28일 입니다.");
          }
}
```
