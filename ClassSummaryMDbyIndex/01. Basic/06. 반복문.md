## 6. 반복문

___

### while문

- 조건식이 참일 경우 조건이 거짓이 될 때까지 해당 블록을 반복하여 수행
- 반복 횟수는 정해져 있지 않지만 특정 조건이 만족될 때까지 루프를 유지할 때 사용

### 기본형태

```java
void format1() {
    while (조건식) {
        // 조건식이 참인 동안 반복해서 실행할 구문
    }
}
```

- 예시

```java
void example1() {
    int i = 0;      // 매개변수 초기화
    while (i < 3) { // 조건 설정
        System.out.println(i);  // 실행할 내용
        i++;        // 매개변수 갱신
    }
}
```

- while문 외부에서 조건식에 사용할 매개변수를 미리 초기화 해야함
- while문 내부에서 매개변수에 대한 갱신이 이루어 져야함
- 조건식을 판별한 후 참일 경우 아래 구문을 수행

### do - while문

- while문과 달리 일단 무조건 한 번은 실행, 이후 조건식을 판별
- 예시

```java
void example2() {
    int i = 10;
    do {
        System.out.println(i);
    } while (i++ < 5);
}
```

### for문

- 조건식이 참일 경우 조건식이 거짓이 될 때까지 해당블럭을 반복하여 수행
- 반복횟수가 명확히 정해져 있을 때 주로 사용

### 기본형태

```java
void format2() {
    for (int i = 0; i < 5; i++) {
        System.out.println(i);
    }
}
```

- 매개변수 초기화, 조건 설정, 매개변수 갱신을 괄호`()`안에 한번에 선언
- 초기화와 조건식, 즘감식은 필요에 따라 생략 가능

### 변형형태 - Enhanced for문

- `배열`내의 모든 원소를 순회할 때 사용하는 형태
- 이 방식으로는 `배열` 내부의 값을 변경할 수 없음

```java
void example3() {
    int[] arr1 = {1, 2, 3, 4};
    for (int elem : arr1) {     // ('데이터 타입' '배열내의 원소를 담을 변수' : '순회할 배열') 형태
        System.out.println(elem);
    }
}
```

### break, continue

- 단독으로 혹은 조건문과 같이 적절히 활용하여 특정한 상황에서 루프를 중단하도록 할 수 있음
- break: 전체 루프를 중단
- continue: 현재 루프를 중단하고 다음 루프로 건너뜀

```java
void example3() {
    for (int i = 0; i < 10; i++) {
        if (i % 2 != 0) continue;   // i가 홀수이면 출력하지 않고 건너뜀
        System.out.println(i);
        if (i == 8) break;      // i가 8이면 전체 루프를 멈추고 for문 탈출
    }
}
```
