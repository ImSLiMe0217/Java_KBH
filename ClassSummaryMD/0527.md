## 0527 진도

### 1. 다차원 배열

- 배열의 요소로 또 다른 배열을 가지는 구조
- 형식:

```java
타입[][] 변수명 = new 타입[행크기][열크기]; //2차원배열
```

- 예시:

```java
void main() {
    // 2x3 행렬
    int[][] matrix1 = new int[2][3];    // 선언 및 생성

    int[][] matrix2 = {                 // 선언 및 생성, 초기화
            {1, 2, 3},
            {4, 5, 6}
    };
}
```

- 다차원 배열은 행마다 열의 길이을 다르게 지정할 수 있음

```java
void main() {
    int[][] matrix2d = new int[3][];
    matrix2d[0] = new int[2];
    matrix2d[1] = new int[3];
    matrix2d[2] = new int[4];
}
```

### 2. 향상된 for문

- 배열 내에 있는 값을 꺼내서 for문의 매개변수로 전달할 때 사용하는 구문
- ArrayIndexOutOfBoundException 예방 가능
- 주의사항
    - 항상 인덱스 순서대로만 꺼낼 수 있음
    - 값을 조회할 때만 사용가능 (수정 불가)
- 형태

```java
void main() {
    int[] arr = new int[] {1, 2, 3, 4, 5};

    int sum = 0;
    for (int i : arr) { // 파이썬을 예로 들면 for (i in list) 형태
        sum += i;       // arr 내의 모든 원소의 합
    }
}
```

### 3. 배열의 복사

- 참조형 데이터 타입의 변수는 new 생성자를 이용하지 않고 생성할 시 같은 내용을 가진 변수가 존재할 경우 해당 변수와 같은 메모리 주소를 저장함
- 예시:

```java
void main() {
    int[] origin = {10, 20, 30};
    int[] shallow = origin;

    // origin과 shallow는 같은 메모리주소를 저장하고 있음

    origin[0] = 100;
    System.out.printf("origin: %d, shallow: %d", origin[0], shallow[0]);
    // origin: 100, shallow: 100
  
  /* 
  origin과 shallow는 같은 주소값을 저장하고 있기 때문에 
  origin에서 변경한 값을 변경하면 shallow에서도 변경된 상태로 보임
  반대의 경우도 동일
   */
}
```

- 위의 경우를 `얕은 복사`라고함

#### 3.1 얕은 복사 vs 깊은 복사

- 얕은 복사
    - 기존의 배열이 있는 주소를 새로운 변수에 저장
    - 한 쪽에서 값을 변경하면 다른 한 쪽에도 그대로 반영됨
- 깊은 복사
    - 기존 배열과 똑같이 생긴 새로운 배열을 생성하고 기존 배열의 내용물을 복사해서 넣는 방식
    - 한 쪽에서 값을 변경해도 다른 한 쪽에서는 반영되지 않음

#### 3.2 여러가지 깊은 복사 방법

- for문을 사용해서 직접 복제

```java
void main() {
    int[] origin = new int[] {10, 20, 30};
    int[] deep = new int[origin.length];
    for (int i = 0; i < deep.length; i++) {
        deep[i] = origin[i];
        System.out.printf("origin: %d, deep: %d\n\n", origin[i], deep[i]);
    }

    deep[0] = 100;
    for (int i = 0; i < origin.length; i++) {
        System.out.printf("origin: %d, deep: %d\n", origin[i], deep[i]);
    }
}
```

- System.arraycopy()
    - 형식: System.arraycopy(원본 배열, 복사 시작점, 대상 배열, 붙여넣기 시작점, 복사 길이)

```java
void main() {
    int[] origin = new int[] {10, 20, 30};
    int[] deep = new int[origin.length];

    // System.arraycopy(원본배열, 복사 시작점, 대상배열, 붙여넣기 시작점, 복사 길이)
    System.arraycopy(origin, 0, deep, 0, origin.length);
    for (int i = 0; i < deep.length; i++) {
        System.out.printf("origin: %d, deep: %d\n\n", origin[i], deep[i]);
    }

    deep[0] = 100;
    for (int i = 0; i < origin.length; i++) {
        System.out.printf("origin: %d, deep: %d\n", origin[i], deep[i]);
    }
}
```

- Array.copyOf()
    - 형식: Array.copyOf(원본배열, 복사할 길이)

```java
void main() {
    origin = new int[] {10, 20, 30};

    // Arrays.copyOf(원본 배열, 복사할 길이)
    deep = Arrays.copyOf(origin, origin.length);

    for (int i = 0; i < deep.length; i++) {
        deep[i] = origin[i];
        System.out.printf("origin: %d, deep: %d\n\n", origin[i], deep[i]);
    }

    deep[0] = 100;
    for (int i = 0; i < origin.length; i++) {
        System.out.printf("origin: %d, deep: %d\n", origin[i], deep[i]);
    }
}
```

- 원본배열.clone()

```java
void main() {
    int[] origin = new int[] {10, 20, 30};
    int[] deep = origin.clone();              // 깊은 복사 (배열 객체 내부메소드)

    for (int i = 0; i < deep.length; i++) {
        deep[i] = origin[i];
        System.out.printf("origin: %d, deep: %d\n\n", origin[i], deep[i]);
    }

    deep[0] = 100;
    for (int i = 0; i < origin.length; i++) {
        System.out.printf("origin: %d, deep: %d\n", origin[i], deep[i]);
    }
}
```

### 4. Arrays 클래스

- 배열 객체를 효율적으로 다룰 수 이는 메서드를 모아놓은 라이브러리 클래스
- https://www.w3schools.com/java/java_ref_arrays.asp

#### 4.1 Arrays.toString()

- 배열 객체를 문자열로 바꿔주는 메소드

```java
void main() {
    int[] arr = {1, 2, 3};

    Arrays.toString(arr); // [1, 2, 3]
}
```

#### 4.2 Arrays.sort()

- 배열의 모든 요소를 오름차순으로 정렬해주는 메소드

```java
void main() {
    int[] arr = {2, 4, 1, 5, 3};

    Arrays.sort(arr); // [1, 2, 3, 4, 5]
}
```

#### 4.3 Arrays.equals()

- 두 배열의 요소들의 값과 순서가 동일한지 체크해주는 메소드

```java
void main() {
    int[] arr1 = {1, 2, 3};
    int[] arr2 = {1, 2, 3};
    int[] arr3 = {2, 3, 1};

    Arrays.equals(arr1, arr2); // true
    Arrays.equals(arr1, arr3); // false
}
```
