## 09. 메서드

- 특정 기능을 수행하기 위해 관련된 코드들을 하나의 블록으로 묶어놓은 것
- 필요할 때마다 호출해서 여러 번 재사용 가능
- 장점:
    - 재사용성 증가
    - 중복제거
    - 유지보수성 향상
    - 가독성 및 구조화

___

### 메서드 형식

- `[접근제어자]` `[반환타입]` `메서드명` `([매개변수 타입] 매개변수)` `{ 기능구현 }`
- 접근제어자:
    - public, default, private 등 메서드의 호출 범위를 제한하는 구역
    - 생략할 경우 기본적으로 default로 적용됨
- 반환타입:
    - void, int, double 등 메서드 종료 후 반환할 값의 타입을 선언하는 구역
    - 반환 타입이 없을 경우 void 사용
- 메서드명:
    - 변수의 식별자처럼 메서드 영역에 저장되어있는 메서드 호출에 사용됨
- 매개변수타입, 매개변수:
    - 메서드 호출 시 내부에서 필요한 값을 인자로 넘겨주기 위한 자리를 만드는 구역
    - 메서드 종료시 메모리에서 모두 지워지기 때문에 메서드 외부에서 사용 불가능
    - 필요 없을 경우 생략가능
    - 여러 매개변수를 선언하는 것도 가능
- 예시

```java
// 배열 내 모든 원소의 평균을 구하는 메서드
double calcAvg(int[] arr) { // int배열을 인자로 받음
    int sum = 0;    // 메서드 내부에서만 사용하는 지역변수, 메서드 외부에서 사용 불가능
    for (int elem : arr) {sum += elem;}
    return sum / (double) arr.length; // double타입의 avg값을 계산해 반환
}
```

### 메서드 호출

- 반환타입이 없는 경우: 메서드명 (인자);
- 반환타입이 있는 경우: [반환되는 변수타입] 변수명 = 메서드명(인자);

```java
void example1() {
    int[] arr = {1, 2, 3, 4, 5};
    double avg = calcAvg(arr);
    System.out.println(avg); // 3.0
}
```

### main 메서드

- 자바 프로그램의 시작점 역할을 하는 특수한 메서드
- JVM은 프로그램을 실행할 때 이 main메서드를 찾아 실행
- main 메서드 형식

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

- 예약어 및 매개변수의 의미
    - public: JVM이 어디서나 프로그램의 main메서드에 접근 가능해야 하기 떄문에 가장 넓은 범위의 제어자를 사요
    - static: JVM이 클래스의 인스턴스 객체를 메모리에 생성하지 않고도 클래스 이름으로 바로 메서드를 호출하여 실행할 수 있도록 함
    - void: main 메서드의 실행이 끝나면 프로그램 자체가 종료되므로, JVM에 반환할 값이 없다는 것을 의미
    - String[] arg: 프로그램 실행시 터미널 등 외부에서 전달하는 문자열 매개변수들을 배열 형태로 전달받음
- 21버전 이후의 자바에서는 public과 static, String[] args는 모두 생략 가능

### 메서드 오버로딩

- 동일한 이름을 가진 메서드를 한 클래스 내에 여러 개 정의하는 것
- 메서드명도 식별자 명명규칙을 따르기에 원래는 같은 이름을 가질 수 없음
- 다른 타입의 매개변수로 같은 기능을 수행할 경우 불필요하게 다른 메서드명을 사용하지 않아도 되도록 함
- 예시:

```java
double calcAvg(int[] arr) {
    int sum = 0;
    for (int elem : arr) {sum += elem;}
    return sum / (double) arr.length;
}

double calcAvg(double[] arr) {
    double sum = 0;
    for (double elem : arr) {sum += elem;}
    return sum / arr.length;
}
```

### 가변인자

- 메서드 호출 시 매개변수의 개수를 동적으로 지정할 수 있게 해주는 기능
- 메서드 정의 시 타입 뒤에 세 개의 마침표(...)를 붙여 선언
- 특징:
    - 내부적으로 배열 처리
    - 가변 인자는 매서드의 매개변수 목록에서 반드시 가장 마지막에 위치해야함
    - 하나의 메서드에는 단 하나의 가변인자만 선언 가능
- 예시:

```java
public void printSum(int... nums) {
    int sum = 0;
    for (int num : nums) {sum += num;}
    System.out.println(nums);
}

void example2() {
    printSum(1, 2, 3);          // 6
    printSum(1, 2, 3, 4, 5);    // 14
}
```
