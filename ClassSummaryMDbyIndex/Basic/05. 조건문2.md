## 4. 조건문 2

___

### switch 문

- 각 case에 맞는 상황일 때만 실행하게 만드는 코드
- 조건문이 참이 되는 범위가 좁거나 1개 씩 띄워진 상황에서 주로 사용

### 기본형태

```markdown
switch (매개변수) {
case (매개변수가 가질 수 있는 값):
실행할 구문
break;
default:
일치하는 경우가 없을 경우 실행할 구문
}
```

- 각 케이스 마지막에 break문이 존재해야함
- 없으면 최초 실행된 케이스 아래있는 케이스들도 모두 실행함

- 예시:

```java
void example1() {
    int floor = 2;
    switch (month) {
        case 1:
            System.out.println("1층");
            break;
        case 2:
            System.out.println("2층");
            break;
        case 3:
            System.out.println("3층");
            break;
        default:
            System.out.println("맞는 경우 없음");
    }
}
```

### 변형형태

자바 12버전부터 도입된 문법

- 화살표 형식
    - 기존의 콜론(`:`)으로 케이스를 선언하던 것 대신 화살표(`->`)를 사용해 선언할 수 있음
    - 이 경우에는 각 케이스마다 break문을 넣을 필요없음
- 예시

```java
void example2() {
    int floor = 0;
    switch (floor) {
        case 1 -> System.out.println("1층");
        case 2 -> System.out.println("2층");
    }
}
```

- 다중 조건 결합
    - 각 case마다 일일히 줄을 바꿔 새로운 case로 선언할 필요없이 쉼표(`,`)를 이용해 선언가능
    - 동일한 결과를 출력하는 case들을 묶어서 선언할 때 사용
- 예시

```java
void example3() {
    int month = 5;
    int lastDay = 0;
    switch (month) {
        case 2 -> lastDay = 28;
        case 4, 6, 9, 11 -> lastDay = 30;
        default -> lastDay = 31;
    }
}
```

- 결과값을 변수에 저장
    - switch문에서 결정되는 값을 변수에 직접 저장가능
    - 이 경우 스위치문을 닫는 중괄호(`}`)옆에 세미콜론(`;`)을 붙여줘야함
- 예시

```java
void example4() {
    int month = 5;
    int lastDay = switch (month) {
        case 2 -> 28;
        case 4, 6, 9, 11 -> 30;
        default -> 31;
    };
}
```
