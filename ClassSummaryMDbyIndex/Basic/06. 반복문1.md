## 6. 반복문 1

___

### while문

- 조건식이 참일 경우 조건이 거짓이 될 때까지 해당 블록을 반복하여 수행
- 반복 횟수는 정해져 있지 않지만 특정 조건이 만족될 때까지 루프를 유지할 때 사용

### 기본형태

```java
void format1() {
    while (조건식) {
        // 조건식이 참인 동안 반복해서 실행할 구문
    }
}
```

- 예시

```java
void example1() {
    int i = 0;      // 매개변수 초기화
    while (i < 3) { // 조건 설정
        System.out.println(i);  // 실행할 내용
        i++;        // 매개변수 갱신
    }
}
```

- while문 외부에서 조건식에 사용할 매개변수를 미리 초기화 해야함
- while문 내부에서 매개변수에 대한 갱신이 이루어 져야함
- 조건식을 판별한 후 참일 경우 아래 구문을 수행

### do - while문

- while문과 달리 일단 무조건 한 번은 실행, 이후 조건식을 판별
- 예시

```java
void example2() {
    int i = 10;
    do {
        System.out.println(i);
    } while (i++ < 5);
}
```
