# JAVA - 객체지향프로그래밍

___

## 객체지향프로그래밍 (기초1)

- 컴퓨터 프로그램을 단순한 명령어의 나열이 아닌 독립된 단위인 객체들이 모인 형태로 작성하는 프로그래밍
- 핵심 지향점
    - 프로그램의 부품화: 데이터와 이를 처리하는 메서드를 객체라는 하나의 부품으로 결합하여 구조화
    - 재사용성: 완성도 높은 객체를 만들어 여러 곳에서 다시 사용
    - 유지보수성: 변경 사항 발생 시 해당 객체만 수정하여 부작용을 최소화

___

### 객체, 클래스, 인스턴스

- `객체`
    - 우리가 인자하고 표현하려는 모든 실체(자동차, 사람, 은행계좌 등)
    - 고유한 속성(상태)과 행위(동작)을 가짐
- `클래스`
    - 객체를 생성하기 위한 설계도
    - 객체의 형태와 기능을 자바 코드로 구현
- `인스턴스`
    - 클래스라는 설계도를 바탕으로 `new`연산자를 사용하여 힙 메모리에 실체화된 결과물

___

### 클래스의 구성 요소

- 필드: 객체의 상태, 속정을 정의한 변수
- 메서드: 객체의 동작, 행위, 기능을 담고 있는 코드 블럭
- 생성자: 인스턴스 생성 시 초기 세팅을 담당
    - `Python`에서의 init과 같은 역할
- 형식

```java
[접근제어자]

class 클래스명 {
    // 필드
    [접근제어자]
    타입 필드명;

    // 생성자
    [접근제어자]

    클래스명(매개변수타입 매개변수명, ...) {
        // 초기 세팅 코드
    }
    
    [접근제어자]

    반환타입 메서드명(매개변수타입 매개변수명, ...) {
        // 실행할 코드
        return 반환값;
    }
}
```

#### 필드

- 클래스 내부에 선언되어 객체의 상태와 속성을 정의하는 변수
- 멤버 변수라고도 부름
- 특징
    - 자동 초기화: 명시적으로 초기화하지 않아도 생성될 때 자료형의 기본값으로 자동 초기화
    - 기본값 목록:
        - 정수형: 0 / 실수형: 0.0 / 논리형: false / 문자형: '\u0000'(공백 문자) / 참조형: null

#### 객체의 생성과 사용

- new 연산자를 사용하여 생성
- 닷(`.`)연산자를 사용하여 객체 내부의 값에 접근 및 메서드 호출

```java
// Person 객체
class Person {
    String name;
    int age;
}

void main() {
    // 인스턴스 생성
    Person p1 = new Person();

    // 필드 접근
    p1.name = "김병호";
    p1.age = 26;
}
```

#### 생성자

- 인스턴스가 생성될 때 가장 먼저 호출되어 인스턴으 변수의 초기 세팅을 담당하는 특수한 메서드
- 형식:

```
[접근제어자] 클래스명(매개변수타입 매개변수명, ...) {
    // 인스턴스 변수 초기화 코드
}
```

- 주요특징:
    - 반환타입이 없고 일반 메서드와 달리 void조차 지정하지 않음
    - 이름은 반드시 소속 클래스명과 대소문자까지 일치해야함
    - 별도로 생성자를 선언하지 않을 경우 기본 생성자가 자동으로 생성됨
    - 별도로 생성자를 선언할 경우 기본 생성자는 생성되지 않으므로 필요 시 별도로 선언해야함
    - 생성자도 메서드이기 때문에 오버로딩 가능

### this와 this()

- `this`
    - 생성된 자기 자신 객체를 가리키는 참조변수
    - 매개변수와 필드명이 같을 때 구별하기 위해 사용
    - `Python`의 self와 유사
- `this()`
    - 같은 클래스 내의 다른 생성자를 호출할 때 사용
    - 생성자 내부의 첫 줄에만 작성가능
- 예시

```java
public class User {
    String name;
    int age;

    public User() {
        this("기본이름", 1);
    }

    public User(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

public class UserTest {
    static void main() {
        User user1 = new User();
        User user2 = new User("김병호", 26);

        System.out.println(user1.name, user1.age);  // "기본이름", 1
        System.out.println(user2.name, user2.age);  // "김병호", 26
    }
}
```
