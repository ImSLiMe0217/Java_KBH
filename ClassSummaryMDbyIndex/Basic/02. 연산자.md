# 자바기초

## 2. 연산자
___
### 연산 규칙
- 연산자 양 쪽에 있는 변수의 타입은 서로 같아야한다.
- 연산자마다 우선순위가 존재한다. ([연산자 우선순위](#연산자-우선순위) 참고)

## 연산자 종류
___
### 산술연산자
- `+`, `-`, `*`, `/`와 같이 수학에서 사용되는 기본 연산자
  - `+`: 두 수의 합
  - `-`: 두 수의 차
  - `*`: 두 수의 곲
  - `/`: 두 수를 나누었을 때의 몫
  - `%`: 두 수를 나누었을 때의 나머지

### 결합연산자
- `+`: 문자열과 함께 사용할 경우 문자열 결합을 의미
```java
void example1() {
    String name = "김병호";
    System.out.println("안녕하세요. " + name + "님"); // 안녕하세요. 김병호님
}
```

### 대입연산자
- `=`: 오른쪽에 있는 '값'을 왼쪽에 있는 '변수'에 저장
- `+=`: 오른쪽에 있는 '값'을 왼쪽에 있는 '변수'의 '값'에 더하고 저장
- `-=`: 오른쪽에 있는 '값'을 왼쪽에 있는 '변수'의 '값'에 빼고 저장
- `*=`: 오른쪽에 있는 '값'을 왼쪽에 있는 '변수'의 '값'에 곱하고 저장
- `/=`: 오른쪽에 있는 '값'으로 왼쪽에 있는 '변수'의 '값'을 나누었을 때의 몫을 저장
- `%-`: 오른쪽에 있는 '값'으로 왼쪽에 있는 '변수'의 '값'을 나누었을 때의 나머지를 저장

### 증감연산자
- `++`, `--`
- 전위형과 후위형으로 구분됨
- 변수의 좌측(전위형)이나 우측(후위형)에 붙여서 사용
- 변수의 값을 1 증가 또는 감소
- 예시:
```java
void example2() {
    int a = 10, b = 10;
    
    // 후위형: 연산 먼저 이후 호출
    System.out.println(++a); // a값 11로 증가, 11 출력
    // 전위형: 호출 먼저 이후 연산
    System.out.println(b++); // 10 출력, b값 11로 증가
}
```

### 비교연산자
- `<` : 좌측이 우측보다 작다
- `<=`: 좌측이 우측보다 작거나 같다
- `>` : 좌측이 우측보다 크다 
- `>=`: 좌측이 우측보다 크거나 같다
- `==`: 좌측과 우측이 같다

### 논리연산자
- `&&`: and
  - 양 쪽 모두 참일 때만 True
  - 이외에는 전부 False
- `||`: or
  - 하나라도 참이면 True
  - 양쪽 모두 거짓일 때만 False
- `!` : not
  - 변수의 왼쪽에 붙여서 사용
  - 참이면 False
  - 거짓이면 True

### 삼항연산자
- `(조건식)` ? `(참일 때 반환값)` : `(거짓일 때 반환값)` 형태
- 예시
```java
void example3() {
    int score = 85;
    score > 60 ? "합격" : "불합격";  // 합격
}
```

## 연산자 우선순위
1. 괄호 `()`
2. 산술연산자 `+` `-` `*` `/` `%`
3. 비교연산자 `<` `<=` `>` `>=` `==` 
4. 논리연산자 `&&` `||` `!`
5. 대입연산자 `=` `+=` `-=` `*=` `/=` `%=`
