## 4. 조건문1

### if 문
- 조건문이 참일 때만 실행하게 만드는 코드
- 조건문이 참인 범위가 넓을 때 주로 사용
___
### if
- if (`조건문`) { 실행할 코드 } 형태로 작성
- 내부에서 사용할 변수는 사전에 초기화 되어있어야 함
- 예시:
```java
void example1() {
    int cond = 5;
    if (cond < 10) {
        System.out.println("조건 만족");
    }
}
```

### if - else
- if문 밑에 else를 붙이고 조건식이 거짓일 경우 실행할 코드를 작성
- 조건식이 거짓일 경우에만 실행되기 때문에 조건식이 참일 경우 실행되지 않음
- 예시: 
```java
void example2() {
    int cond = 0;
    if (cond > 10) {
        System.out.println("조건 만족");
    } else {
        System.out.println("조건 불만족"); // 이게 출력됨
    }
}
```

### if - else if - else
- if문 밑에 else if를 붙이고 if문에서와는 다른 조건을 만족 할 경우 실행할 코드를 작성
- else는 필요없으면 사용 안해도 되지만 나올 때는 항상 마지막에만 나와야 함
- 예시: 
```java
void example3() {
    int cond = 15;
    if (cond > 20) {
        System.out.println("조건1 만족");
    } else if (cond > 10) {
        System.out.println("조건2 만족");
    } else {
        System.out.println("조건 불만족");
    }
}
```
