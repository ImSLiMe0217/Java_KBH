## 복습

### 1. 반복문

#### 1.1 while문
```java
void main() {
    int i = 0;
    int sum = 0; // 결과값을 저장할 변수를 미리 초기화
    while (i <= 10) {
        sum += i++;
            }
}
```
- 조건식 판별 -> 참일 경우 구문 실행 -> 조건식 판별 -> 조건식이 거짓이 될 때까지 반복
- 내부에서 사용되는 변수를 사전에 초기화 해두어야 함
- 반복 횟수가 정해져 있지 않을 때 주로 사용

#### 1.2 do - while문
```java
void main () {
    int i = 0; 
    int sum;    // 결과값을 저장할 변수를 초기화 안해도 사용 가능
    do {
        sum += i++;
    } while (i <= 10);
}
```
- 구문 실행 -> 조건식 판별 -> 조건식이 참일 경우 구문 실행 / 거짓일 경우 종료
- while과 다르게 내부에서 사용되는 변수를 미리 초기화 해 둘 필요 없음

#### 1.3 for문
```java
void main () {
    int sum = 0;
    for (int i = 0; i <= 10; i++) {
        sum += i;
    }
}
```
- 매개변수 초기화와 갱신, 종료조건 설정을 ()안에 한 번에 선언 가능
- 반복 횟수가 정해져 있을 경우 주로 사용
