## 복습

### 1. 조건문

```java
void main () {
  if (조건문1) {
      // 조건문1이 참일 때 실행할 구문
          } else if (조건문2) {
      // 조건문2가 참일 때 실행할 구문
          } else {
      // 모든 조건이 거짓일 때 실행할 구문
          }   
        }
```
- else if는 여러번 사용가능
- else는 없거나 1번만 사용


### 2. switch
```java
void main () {
  switch (매개변수) {
      case 가능한_변수의_값:
          // 해당 케이스에서 실행할 구문
          break; // 실행 후 switch문 탈출
      default:
          // 해당하는 케이스가 없을 경우 실행하는 구문
          // break 없어도 됨
          }
}
```

- 14버전 이후 도입된 새로운 switch문법
  - 화살표 ( `->` ) 문법 사용 시 break문 필요 없음 (case 조건 -> 구문)
  - 다중 조건 결합: 쉼표 (`,`)를 사용해서 여러 case 조건을 한 줄에 묶어 선언
  - switch문의 최종 결과를 변수의 값으로 저장 가능

## 0521 진도

### 1. 반복문

#### 1.1 while문
- 지정한 조건식이 참일 동안 해당 블록을 반복하여 수행
- 형식: 
```java
void main () {
  while (조건식) {
      // 조건식이 참인 동안 반복해서 실행할 구문
          }
        }
```

#### 1.2 do-while문
- 일반 while문은 조건식을 판별하고 참일 경우에만 실행 -> 한 번도 실행 안할 수 있음
- 일단 무조건 1번은 실행하고 조건식을 판별함
- 형식: 
```java
void main () {
  do {
      // 실행할 구문
  } while (조건식);   
}
```

#### 1.3 for문
- 지정한 조건식이 참일 경우 조건이 거짓이 될 때까지 해당 블록을 반복수행
- 반복 횟수가 명확할 때 주로 사용
- 형식:
```java
void main () {
  for (매개변수 초기화; 조건식; 증감식) {
      // 조건식이 참일 동안 실행할 구문
          }   
}
```
